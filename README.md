# Linux-System-Recon
A bash script to gather basic system information

### About the Tool
This is a basic system enumeration script written in Bash. 
It automates the process of gathering essential system information, which is a crucial first step in any local privilege escalation or system audit.

**Features:**
* Logs the current Date & Time
* Identifies logged-in users
* Retrieves Network Interfaces & IP addresses
* Pulls OS and Kernel version details

### Usage
```bash
chmod +x sys_recon.sh
./sys_recon.sh
'''
